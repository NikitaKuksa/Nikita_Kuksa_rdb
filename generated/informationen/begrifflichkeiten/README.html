Schritte:

  1. Den Explorer in VS Code öffnen (Symbol ganz oben links in der Seitenleiste, sieht aus wie zwei Seiten).
  2. Im Explorer die Datei generated/informationen/begrifflichkeiten/stichwortverzeichnis_relationale_datenbanken.html suchen und mit einem Rechtsklick darauf klicken.

  4. Der Browser öffnet sich automatisch mit der Suchseite.
  5. Im Suchfeld oben einfach einen Begriff eingeben, z. B. JOIN, 3NF oder Anomalie.
